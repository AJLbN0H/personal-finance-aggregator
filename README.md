#  Приложение для анализа транзакций

## ```views```
Принимает на вход строку с датой и временем в формате YYYY-MM-DD HH:MM:SS и возвращает JSON-ответ со следующими данными:
* Приветствие в формате "???", где ??? — «Доброе утро» / «Добрый день» / «Добрый вечер» / «Доброй ночи» в зависимости от текущего времени.
* По каждой карте:
    * последние 4 цифры карты;
    * общая сумма расходов;
    * кешбэк (1 рубль на каждые 100 рублей).
* Топ-5 транзакций по сумме платежа.
* Курс валют.
* Стоимость акций из S&P500.

### Тестирование ```views```
* Проверка правильности выполнения функции приветствия в зависимости от текущего времени;
* Проверка правильности обработки некоректных данных;
* Тест получения списка 'Топ 5 Лучших транзакций';
* Проверка разных случаев работы функций;

## ```utils```
В нем находяться вспомогательные функции для ```views.py```

### Тестирование ```utils```
* Проверка получение DataFrame из xlsx файла;
* Проверка корректного получения настроек пользователя;
* Тестиирование корректности декораторов;
* Проверка разных случаев работы функций;

## ```services```
Возвращает JSON со всеми транзакциями, которые относятся к переводам физлицам

### Тестирование ```services```
* Проверка работоспособности функций;
* Проверка обработки случаев с некоректными данными;

## ```reports```
Возвращает траты по заданной категории за последние три месяца (от переданной даты)

### Тестирование ```reports```
* Проверка работоспособности функций;
* Проверка обработки случаев с некоректными данными;
